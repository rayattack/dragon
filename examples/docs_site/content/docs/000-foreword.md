# Foreword

This little book is served by the Dragon program sitting next to it. The page
you are reading was a Markdown file a moment ago; a renderer written in Dragon
turned it into HTML, a highlighter written in Dragon colored the code blocks,
and the standard library's HTTP server sent it to your browser.

> The site is the dogfood. When something here is awkward to build, the fix
> belongs in the language or the standard library - not in a workaround.

Turn the page to *Hello, Dragon* to see the smallest program, then follow the
sidebar through types, lists, and loops.
